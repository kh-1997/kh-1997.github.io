# LeetCode-二分查找

------
#### 解题思路

- 二分查找查找一个数

- base条件：left<=right



```
class Solution {
    public int binarySearch(int[]nums,int target)
        int left = 0;
        int right = nums.length-1;
        while(left<=right){
            int mid = left+(right-left)/2;
            if(nums[mid]==target){
                return mid;
            }else if(nums[mid]<target){
                left = mid+1;
            }else if(nums[mid]>target){
                right = mid-1;
            }
        }
        return -1;
}
```

- 二分查找，查找左边区间

```
int leftBound(int[]nums,int target){
     int left = 0;
     int right = nums.length-1;
     while(left<=right){
         int mid = left+(right-left)/2;
         if(nums[mid]==target){
             right = mid -1;
         }else if(nums[mid]<target){
             left = mid+1;
         }else if(nums[mid]>target){
             right = mid-1;
         }
      }
      if(left>=nums.length||nums[left]!=target)
      	return -1;
      return left;
}
```
